Deprecated versions
-------------------

Older releases can be download from the `GitHub repository
<https://github.com/dftbplus/dftbplus/releases/>`_.

**Note:** These versions of DFTB+ has been **deprecated** as a newer version is
available now. There will be no bug fixes or any kind of support provided for
any of these versions.

**Please use the** :ref:`current stable version <sec_download_stable>` **instead!**
