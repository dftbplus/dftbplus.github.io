Documentation

Recipes
-------

Have a look at the recipes, if you are new to DFTB+ (it contains tutorials /
introduction to several topics), or if you look for an overview about a specific
topic or for some useful tricks.

* `DFTB+ recipes (html) <http://dftbplus-recipes.readthedocs.io/>`_
