This is the github.io project page for the DFTB+ project, a fast semi-empirical code for quantum chemistry, materials science, bio-science and condensed matter simulations.

This consists of the

  • The DFTB+ code itself
  • Recipes giving examples for calculating some systems with DFTB+
  • The MPIFX and SCALAPACKFX libraries
  • The project's developer guide

The DFTB+ developer group is hosted on github.
